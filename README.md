# Inventory Management System
This project is a web-based application developed for our final requirement. It is built using Java and Spring Boot to handle basic inventory operations. The system allows users to manage product records, track available stock in real-time, and process sales deductions.

Project Objectives:
- Provide a straightforward interface for adding new products to the database.
- Track inventory levels accurately after every transaction.
- Implement an automated low-stock alert when items reach critical levels.
